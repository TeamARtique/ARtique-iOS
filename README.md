# 💜 ARtique
> **🖼 언제 어디서나 즐길 수 있는 AR 전시 플랫폼 'ARtique'**  
>
> 서울여자대학교 디지털미디어학과 졸업프로젝트 <br>
> 프로젝트 기간 : 2021.09 ~ 2022.06 <br> <br>
> ARtique에서는 누구나 작가가 되어 자신의 작품을 전시할 수 있으며, <br>
> 나의 평면 작품들이 전시공간과 함께 AR 전시회로 재탄생하게 됩니다. 
>
